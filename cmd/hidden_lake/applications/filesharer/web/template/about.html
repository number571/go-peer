{{if (eq .FLanguage 0)}} The HLF is a file sharing service based on the Anonymous Network Core (HLS) with theoretically provable anonymity. A feature of this file sharing service is the anonymity of the fact of transactions (file downloads), taking into account the existence of a global observer. {{else if (eq .FLanguage 1)}} HLM - это мессенджер, основанный на ядре анонимной сети (HLS) с теоретически доказуемой анонимностью. Особенностью данного файлообменника становится анонимность факта совершения транзакций (скачивания файлов) с учётом существования глобального наблюдателя. {{else if (eq .FLanguage 2)}} HLF estas dosierpartumado servo bazita sur La Anonima Reto Kerno (hls) kun teorie pruvebla anonimeco. Karakterizaĵo de ĉi tiu dosierpartuma servo estas la anonimeco de la fakto de transakcioj (dosieraj elŝutoj), konsiderante la ekziston de tutmonda observanto. {{end}}

{{if (eq .FLanguage 0)}} The HLS is the core of an anonymous network with theoretically provable anonymity, built on the basis of friend-to-friend (F2F) connections. It is based on the "queues" model. In certain implementations, it may exhibit the property of "abstractness". {{else if (eq .FLanguage 1)}} HLS - это ядро анонимной сети с теоретически доказуемой анонимностью, построенной на основе соединений "друг с другом" (F2F). Ядро основано на модели "очередей". В определенных реализациях ядро HLS может обладать свойством "абстрактности". {{else if (eq .FLanguage 2)}} HLS estas la kerno de anonima reto kun teorie pruvebla anonimeco, konstruita sur la bazo de ligoj "unu kun la alia" (F2F). La kerno estas bazita sur "vico" modelo. En certaj kazoj En efektivigoj, LA hls-kerno povas havi la posedaĵon de "abstrakteco". {{end}}
