> [!CAUTION]
> There is no asymmetric cryptography in this version of the library. The Hidden Lake network is based only on symmetric encryption and cryptographic hash functions. Thus, the HL network on this `quantum` branch is able to function successfully even in post-quantum time. The branch is currently being tested...

Examples: `examples/anonymity/echo_service/routing` and `examples/anonymity/echo_service/secret_channel/common`
